# Project Introduction
웹 애플리케이션 back-end를 구성 하였습니다.
## Auth
* 회원가입
* 로그인
## Board
* 최신 게시물 목록
* 주간 상위 게시물 목록
* 검색 게시물 목록
* 특정 유저 게시물 목록
* 게시물 상세/작성/수정/삭제
* 좋아요 기능/목록
* 댓글 작성/목록
## Search
* 인기 검색어 목록
* 관련 검색어 목록
## User
* 유저 정보
* 로그인 회원 정보
* 닉네임 수정
* 프로필 이미지 수정
